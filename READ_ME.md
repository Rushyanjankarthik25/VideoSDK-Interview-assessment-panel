# VideoSDK Interview Pipeline

## Overview

This project implements a full-stack, event-driven interview intelligence pipeline using VideoSDK, Node.js, MongoDB, and Gemini AI.

---

## Features

- Real-time video interviews
- Event-driven pipeline
- Transcription + AI scoring
- Structured report generation
- Pipeline observability API

---

## Tech Stack

- Frontend: Next.js 15, React 19, TypeScript
- Backend: Node.js, Express
- Database: MongoDB
- Queue: BullMQ + Redis
- AI: Google Gemini

---
